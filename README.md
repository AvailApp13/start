# FelixDV landing

Готовый статический лендинг для GitHub + Vercel.

## Что внутри
- `index.html` — основной лендинг
- `styles.css` — стили
- `script.js` — анимации, форма, копирование контактов, валидация телефона `+7`
- `assets/` — логотип, визуалы работ и блок отзывов
- `vercel.json` — минимальная конфигурация для Vercel

## Как загрузить
1. Распакуйте архив
2. Загрузите файлы в корень репозитория GitHub
3. Если репозиторий уже подключен к Vercel, деплой обновится автоматически

## По форме
Форма работает без отдельного сервера:
- WhatsApp: открывает подготовленное сообщение
- Telegram: копирует текст заявки и открывает окно шаринга
- MAX: копирует текст и номер
- Сотовая связь: копирует текст заявки и запускает звонок
